# EDA_AmazonPrimeVideo
Exploratory Data Analysis on Amazon Prime Video dataset

## Project Summary
This project analyzes over 9,800 titles from Amazon Prime Video, including Movies and TV Shows. It explores content trends, popularity, ratings, and audience preferences.

## Dataset
- `titles.csv` – Contains title info like name, type, genres, release year, ratings, etc.
- `credits.csv` – Contains actors, directors, and character info.

## Libraries Used
- Pandas, NumPy, Matplotlib, Seaborn

## Key Insights
- Movies and TV Shows distribution
- Most popular genres and top-rated titles
- Correlation between IMDb score, TMDB score, votes, and popularity

## How to Run
Open the notebook in Google Colab and run all cells to see the analysis and plots.
